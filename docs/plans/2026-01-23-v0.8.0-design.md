# v0.8.0 Design Document

**Date:** January 23, 2026
**Version:** 0.8.0
**Status:** Approved

---

## Overview

v0.8.0 introduces a color theming system, education content, and template expansion to address the core problem: Zoho employees have outdated signatures with old logos and poor formatting that reflect poorly on the brand.

### Goals
1. Educate employees on signature importance
2. Allow color customization within brand guidelines
3. Expand template options to 6 styles with role-specific guidance
4. Maintain 100% brand compliance (Zoho colors only)

### Out of Scope
- Event banner functionality (deferred to v0.9.0)
- Profile photo support (future consideration)
- Auto-population from Zoho services

---

## Design Decisions

### Education Content
**Problem:** Employees don't understand why signature quality matters
**Solution:** Replace generic tagline with directive message

**Current text:**
> "Create a professional email signature for your Zoho account"

**New text:**
> "Outdated signatures reflect badly on Zoho. Use this tool for professional, brand-compliant email signatures."

**Rationale:** Direct, problem-focused messaging (2 lines, ~15 words) that emphasizes brand impact.

---

### Color Switcher System

**Research basis:** [2026 email signature trends](https://www.crossware365.com/blog/email-signature-design-trends-for-2026-whats-in-and-whats-out) show personalization within brand guidelines is preferred.

**Official Zoho Brand Colors:** (from https://www.zoho.com/branding/)
- **Red:** #E42527
- **Green:** #089949
- **Blue:** #226DB4
- **Yellow:** #F9B21D

**UI Design:**
- Location: Preview section, below style selector
- Format: 4 rectangular buttons (80px × 40px)
- Style: Colored backgrounds with white text labels
- Selected state: 3px solid border (#333)
- Default: Red (#E42527)

**Color Application (Middle Ground Approach):**
- ✅ Separator bullets/lines
- ✅ Vertical bars (Modern template)
- ✅ Social media icon backgrounds
- ✅ Link colors
- ❌ Name text (stays black for readability)

**State Management:**
- Store in `AppState.accentColor`
- Persist to `localStorage.setItem('signature-accent-color', color)`
- Apply via CSS custom property `--accent-color`

---

### Template Expansion (4 → 6)

**Research findings:**
- [C-Suite signatures](https://www.wisestamp.com/signature-examples/ceo-and-executives-email-signatures/): "Less is more" approach maintains authority
- [Sales signatures](https://blog.hubspot.com/sales/sales-email-signature): More complex, CTA-focused
- [Customer Support](https://www.wisestamp.com/templates/professionals/customer-support/): Simple, functional, fits in ticket replies
- [Marketing signatures](https://exclaimer.com/email-signature-handbook/professional-email-signature-examples/): Showcase creativity

**New Templates:**

#### Template 5: "Executive"
**Target:** VPs, C-Suite, Senior Leadership

**Design characteristics:**
- Larger name (20px font, bold)
- Center-aligned (formal symmetry)
- Minimal contact info (email, phone, website only)
- No social media links
- Logo below name (40px height)
- Subtle accent line below name (1px horizontal rule)
- Generous whitespace (12px spacing vs. 8px)

**Rationale:** Research shows executives need clean, authoritative signatures without social media clutter.

#### Template 6: "Bold"
**Target:** Marketing, Events Team

**Design characteristics:**
- Name/title on colored background block (uses accent color)
- White text on colored block (WCAG AA compliant: 4.5:1 contrast)
- Padding: 12px vertical, 16px horizontal
- Logo left-aligned (32px)
- Horizontal contact layout
- Prominent social media (accent colored backgrounds)
- 8px border radius on colored block

**Rationale:** Marketing/Events teams need eye-catching signatures that showcase brand personality.

---

### Department Recommendations

**Change:** Replace generic descriptions with role-specific guidance.

**Research basis:** Organizations assign [templates by department/role](https://www.hubspot.com/email-signature-generator/email-signature-management) for consistency.

**Mapping:**

| Template | Recommendation | Rationale |
|----------|---------------|-----------|
| **Classic** | Executives, HR, Corporate Comms | Traditional, formal leadership style |
| **Compact** | Customer Support | Fits in ticket replies, simple/functional |
| **Modern** | Sales, Presales Engineers | Professional first impression, CTA emphasis |
| **Minimal** | Anyone | Universal minimalist (2026 trend) |
| **Executive** | VPs, C-Suite, Senior Leadership | Clean, authoritative, maintains gravitas |
| **Bold** | Marketing, Events Team | Showcases creativity with color |

**Display:** Always visible under template name (replaces current `.style-desc` text).

#### 4. app.js

**New state management:**
```javascript
const AppState = {
    // ... existing properties
    accentColor: '#E42527'  // Add accent color
};

// Load from localStorage on init
function initializeApp() {
    const savedColor = localStorage.getItem('signature-accent-color');
    if (savedColor) {
        AppState.accentColor = savedColor;
        document.querySelector(`[data-color="${savedColor}"]`).classList.add('selected');
    }
}

// Color switcher event listeners
document.querySelectorAll('.color-btn').forEach(btn => {
    btn.addEventListener('click', (e) => {
        const color = e.target.dataset.color;

        // Update state
        AppState.accentColor = color;

        // Update UI
        document.querySelectorAll('.color-btn').forEach(b => b.classList.remove('selected'));
        e.target.classList.add('selected');

        // Persist
        localStorage.setItem('signature-accent-color', color);

        // Update preview
        updatePreview();
    });
});
```

## Accessibility Considerations

### Color Switcher
- ✅ Text labels on buttons (not icon-only)
- ✅ Focus ring on keyboard navigation
- ✅ ARIA labels: `aria-label="Select red accent color"`
- ✅ Selected state visible (3px border, not color-only)

### Yellow Button Text
- ✅ Dark text (#333) on yellow background for contrast
- ✅ Meets WCAG AA: 4.5:1 ratio

### Bold Template Colored Block
- ✅ Must validate contrast ratios for all 4 colors
- ✅ Red: 4.52:1 (WCAG AA Pass)
- ✅ Green: 4.54:1 (WCAG AA Pass)
- ✅ Blue: 4.51:1 (WCAG AA Pass)
- ✅ Yellow: Use dark text instead of white
